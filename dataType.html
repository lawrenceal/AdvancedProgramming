基本数据类型： Undefined Null Number String Boolean Object

typeof： undefined number string boolean object function

undefined: 没有必要显示设置，未初始化变量就是undefined， 未经声明的变量用typeof测试类型也是undefined， 未声明的变量使用会报错，（除变量声明提升，变量声明提升会输出undefined） 在严格模式下给未声明的变量赋值会报错 e.g. a = 1

null: 空指针对象

boolean: true false，true不一定等于1，false不一定等于0，所有的数据类型都可以转换为boolean类型，可以通过Boolean()转型函数实现。
数据类型 转化为true的值 转化为false的值
boolean true false
string 非空字符串 ""
number 任何非0数值 0 和 NaN
object 任何对象 null
undefined n/a undefined


number: 整数和浮点数， 整数支持八进制（0开头），十进制，十六进制（0x开头 a-f A-F 不区分大小写）。计算时都会转换为十进制。 浮点数必须包含一个小数点，小数点后必须有一位数字。(.1 不推荐)， 由于保存浮点数所需的内存空间是保存整数的两倍，所有ES会不失时机的将浮点数转换为整数 e.g. var a = 1.0 支持科学技术表示法 2e10 0.2e-10，浮点数的精度可达到17位，但是浮点型在计算时会有精度损失 e.g. 0.1 + 0.2 = 0.300000000 ...4 Number.MIN_VALUE最小值， Number.MAX_VALUE最大值，超出范围正无穷Infinity，负无穷-Infinity， isFinite()判断是否超出范围 NaN not a number， 0 / 0 = NaN， 2 / 0 = infinity，任何涉及到NaN的操作都会返回NaN，NaN和谁都不相等，包括NaN自己，isNaN会进行数值转换。
数值转换： Number parseInt parseFloat Number可以处理任何的数据类型。 parseInt， parseFloat专门把字符串转换为数字(也可以转换，转换结果根据规则为NaN) Number("") //0
Number("false") //NaN
Number("123") //123
Number(true) //1
Number("010") //忽略前导0
Number(null) //0
Number(undefined) //NaN
Number("0x12") //18能够解析16进制
Number如果处理Object会先调用对象的valueOf()，然后按照以上规则处理，如果结果是NaN，则调用对象的toString()方法，继续按照以上的规则转换返回的字符串。
parseInt会忽略前导空格，找到字符第一个非空字符，如果第一个字符不是数字或者负号则返回NaN。如果符合规则继续解析，直到遇到非数字字符。 parseInt能够识别各种格式的整数字符（八进制，十进制，十六进制），parseInt在解析八进制时候ES3和ES5有区别，ES5会按照十进制去解析，所以需要显示的给出第二个参数（进制）。
parseInt("") //NaN
parseInt("010") //es3 = 8 es5 = 10
parseInt(" 12") //12
parseInt("0null") //0
parseInt("0x12") //18
parseInt("AF") //NaN
parseInt("AF", 16) //175
建议在无论什么情况下都明确的指定基数
parseFloat与parseInt的解析方式类型，从第一个非空字符解析，知道遇到一个无效的浮点字符为止，第一个小数点是有效的，第二个小数点无效的。会一直忽略前导0，只能解析10进制
parseFloat(".1") //.1
parseFloat("010") //10
parseFloat("0x12") //0
parseFloat("12e-2") //0.12
parseFloat("12eaf") //12

String 有0或多个16位Unicode字符组成字序列。字符串是不可变的，一旦创建，他们的值就是不能改变。
var lang = "java"; lang = lang + "script";
首先创建一个能容纳10个字符的新字符串，然后在这个字符串中填充"Java"和"script"。然后销毁原来的"java"和"script"
